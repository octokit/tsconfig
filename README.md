## Usage

<table>
<tbody valign=top align=left>
<tr><th>

Browsers

</th><td width=100%>

`@octokit/tsconfig` is not meant for browser usage.

</td></tr>
<tr><th>

Node

</th><td>

Install with `npm install @octokit/core @octokit/tsconfig`

```js
const { config } = require("@octokit/tsconfig");
```

</td></tr>
</tbody>
</table>

```js
{
  "extends": "@octokit/tsconfig"
}
```
